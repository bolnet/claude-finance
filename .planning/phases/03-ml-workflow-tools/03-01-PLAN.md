    Create src/finance_mcp/tools/csv_ingest.py with the following exports:
      - `ingest_csv(csv_path: str, target_column: str = "") -> str` — public MCP tool
      - `_detect_structure(df: pd.DataFrame) -> dict` — internal helper (testable)
      - `_clean_dataframe(df: pd.DataFrame, numeric_cols: list) -> pd.DataFrame` — internal helper (testable)

    Implementation of ingest_csv:
    1. Validate path: if not os.path.exists(csv_path) → raise ToolError with message "The file '{csv_path}' was not found. Check the path and try again."
    2. Read CSV: df = pd.read_csv(csv_path)
    3. Detect structure using _detect_structure: numeric_cols via select_dtypes(include=[np.number]), cat_cols via select_dtypes(include=['str']), missing count per column
    4. Call _clean_dataframe on all numeric_cols to remove IQR outliers (1.5*IQR rule) and fill remaining missing numeric values with column median. Use df.copy() — never mutate original (immutability rule).
    5. EDA charts: for each numeric column (up to 4 columns to keep chart count reasonable), create a matplotlib histogram using plt.subplots() and save via save_chart(fig, f"eda_{col_name}.png"). Collect all saved paths into chart_paths list.
    6. Build plain_english summary: "Loaded {N} rows and {M} columns from {basename}. Found {len(numeric_cols)} numeric and {len(cat_cols)} categorical columns. Cleaned {removed_rows} outlier rows. {missing_summary}."
    7. Build data_section: df.describe().to_string() on cleaned data
    8. Return format_output(plain_english, data_section, chart_paths)

    Implementation of _clean_dataframe:
    - df = df.copy()
    - For each col in numeric_cols: compute Q1, Q3, IQR; filter rows to [Q1 - 1.5*IQR, Q3 + 1.5*IQR]; use df.copy() after each filter to avoid chained assignment warning
    - For each col in numeric_cols: df[col] = df[col].fillna(df[col].median())
    - Return cleaned df
