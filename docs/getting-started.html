Get Started

Start analyzing finance data in minutes — no Python expertise required.

Path A: Claude Code (recommended)

Best if you use Claude from the terminal or your code editor

1
Clone the repository

Download the finance skill to your computer. This creates a folder called Claude-Finance.

git clone https://github.com/bolnet/Claude-Finance.git
cd Claude-Finance
2
Install the finance tools

This installs everything the finance server needs — chart libraries, data feeds, and machine-learning models.

pip install -e .
3
Open Claude Code

Claude Code automatically finds the finance server when you start it from the project folder. No configuration file to edit.

claude
4
Verify it works

Type the command below. You should see a confirmation that the finance skill is active.

/finance

Path B: claude.ai (browser)

Best if you use Claude in the browser at claude.ai

1
Clone and install

Download the finance skill and install everything it needs.

git clone https://github.com/bolnet/Claude-Finance.git
cd Claude-Finance
pip install -e .
2
Start the finance server

This starts a local web server that claude.ai can connect to. Leave this terminal window open — the server must stay running.

python -m finance_mcp.server_http

The server will listen on port 8000. You will see a message confirming it is running.

3
Create a public tunnel with ngrok

ngrok makes your local server reachable from the internet so claude.ai can connect. Open a new terminal window and run these commands in order.

brew install ngrok
ngrok config add-authtoken YOUR_TOKEN
ngrok http 8000

Sign up for a free account at ngrok.com to get your token. ngrok will show a public URL like https://abc123.ngrok-free.app — copy it.

Heads up: the ngrok URL changes every time you restart ngrok. You will need to update the URL in claude.ai Settings each time. To avoid this, claim a free static domain at dashboard.ngrok.com/domains (one per account).
4
Add the server to claude.ai

Go to claude.ai, open Settings, then Integrations, click Add Integration, and paste the ngrok URL you copied.

5
Verify it works

Start a new conversation and type the command below. You should see the finance skill respond.

/finance

Your First Analysis

Once you see the finance skill is active, try this prompt:

Show me a 6-month price comparison of Apple, Google, and Microsoft

Claude will pull live market data, generate a chart, and explain the trends — all in plain English.

Troubleshooting

Charts fail with a network error

Yahoo Finance limits how fast you can pull data. Wait 30 seconds and try again.

'python' not found

On macOS, try python3 instead of python in every command above.

ngrok shows 'authentication required'

Sign up for a free account at ngrok.com, then run ngrok config add-authtoken YOUR_TOKEN with the token from your ngrok dashboard.
